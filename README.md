### About The Project

A simple react-redux movie web app built using the OMDb API

### Usage

Enter the name of a movie or TV series into a text input and receive a list of movies.

### Installation

```sh
npm install
npm start
```

### Built With

- [React](https://reactjs.org/)
- [Redux](https://redux.js.org/)
